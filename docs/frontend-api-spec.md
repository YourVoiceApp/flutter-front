# Frontend API Spec

기존 단일 문서 경로를 유지하면서, 실제 내용은 화면 기준 문서로 분리한 버전을 안내합니다.

## 시작 위치

- 메인 인덱스: `docs/frontend-api/index.md`

## 화면별 문서

- 인증: `docs/frontend-api/auth.md`
- 음성 화면: `docs/frontend-api/voice-screen.md`
- 함께 화면: `docs/frontend-api/rooms-screen.md`
- 마켓 화면: `docs/frontend-api/market-screen.md`
- 마이페이지: `docs/frontend-api/mypage-screen.md`

## 참고

- 공통 정보와 전체 상태 요약은 `docs/frontend-api/index.md` 에 정리되어 있습니다.
- 세부 API 예시와 프론트 연동 메모는 각 화면 문서에 분리되어 있습니다.
- 프론트 피드백 기준으로 `연결 가능`, `아직 데모`, `백엔드 blocker` 상태를 각 파일에 따로 적어두었습니다.
