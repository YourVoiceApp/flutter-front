# Backend Todo Docs

프론트 연동 과정에서 확인된 백엔드 추가 작업 / 계약 보완 사항을 정리하는 폴더입니다.

## 문서 목록

- `backend-api-handover.md`
  - 프론트 기준으로 백엔드에 요청해야 하는 API / 응답 필드 / 계약 보완 사항

## 사용 목적

- 프론트에서 이미 붙은 API와 아직 막혀 있는 API를 구분
- 백엔드에 바로 전달할 수 있는 TODO 목록 유지
- 문서상 명세와 실제 프론트 구현 사이의 간극 기록

## 작성 원칙

- 우선순위를 `P0`, `P1`, `P2`로 구분
- "왜 필요한지"를 프론트 화면 기준으로 설명
- 가능하면 요청/응답 예시를 같이 적기
- 기존 API 수정이 필요한 경우, 어느 응답 필드가 부족한지 명확히 적기
